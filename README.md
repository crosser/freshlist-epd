# ESP32 app that fetches some data over http and displays on e-paper display

## What is this?

A reimplementation of
[this project](https://git.average.org/cgit/freshlist.git/)
([mirror](https://github.com/crosser/freshlist)), but unsing 5.83"
e-paper display from waveshare.

Run `idf.py mkconfig` to enter your WiFi SSID and WPA password (only WPAn
is supported), url and timezone.
